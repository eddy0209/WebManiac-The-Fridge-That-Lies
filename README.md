# 🧊 ShelfSense

### Smart Pantry Intelligence for Modern Households

ShelfSense is a modern pantry and grocery management platform designed to help users track inventory, monitor stock levels, and maintain visibility over household groceries through a clean and intuitive interface.

Built using React, Supabase Authentication, and Vite, ShelfSense combines inventory tracking, activity monitoring, and future-ready multi-user collaboration features.

---

## 🚀 Live Demo

https://shelve-sense.vercel.app/

---

## ✨ Features

* 📦 Add, consume, and restock pantry items
* 📊 Real-time inventory health monitoring
* ⚠️ Automatic low-stock and empty-item detection
* 📜 Activity timeline for inventory actions
* 🔐 Google Authentication with Supabase
* 🎨 Modern glassmorphism-inspired UI
* 📱 Responsive design
* ⚡ Smooth animations with Framer Motion
